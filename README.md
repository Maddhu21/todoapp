# ToDoApp
This project is a small project that I made while learning Laravel 12. 
It entails basic features such as:
1. Register
2. Login
3. Create Task
4. Update Task Status
5. Update Task Details
6. Delete Task
7. Master Records
   7.1 Status Master
   7.2 Role Master
   7.3 Test Master ( For Testing Purposes )
